--- layout: default title: "MakeTables Documentation" --- # MakeTables A Python package for creating publication-ready tables from statistical models and descriptive statistics. ## Quick Start ### Installation ```bash pip install maketables ``` ### Basic Usage ```python import maketables as mt import pandas as pd # Load data df = mt.import_dta("https://www.stata-press.com/data/r18/auto.dta") # Create descriptive statistics table mt.DTable(df, vars=["mpg","weight","length"], bycol=["foreign"]) ``` ## Features - **Multiple Output Formats**: Generate tables as Great Tables (HTML), LaTeX, or Word documents - **Statistical Integration**: Native support for statsmodels, pyfixest, and linearmodels - **Flexible Formatting**: Customizable labels, notes, significance levels, and styling - **Data Import**: Built-in Stata file (.dta) import/export with variable label preservation ## Navigation - [Examples and Tutorials](examples.html) - [API Documentation](api.html) - [GitHub Repository](https://github.com/dsliwka/maketables)
